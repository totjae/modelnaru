# 모델나루 텍스트 첨부 수동 시험

## 1. 파일별 단독 인식

각 파일을 하나씩 첨부하고 다음 문구를 전송한다.

> 첨부한 파일만 근거로 답해줘. 문서 식별자 또는 marker를 먼저 쓰고, 파일의 핵심 내용을 3개 항목으로 정리해줘. 파일에 없는 내용은 추측하지 마.

예상 식별자:

- `01-basic-korean.txt`: `MN-TXT-7429`
- `02-meeting-notes.md`: `MN-MD-VIOLET-381`
- `03-products.json`: `MN-JSON-ORANGE-552`
- `04-server-metrics.csv`: `MN-CSV-CYAN-904`
- `05-events.jsonl`: `MN-JSONL-BLUE-117`
- `06-follow-up-memory.yaml`: `MN-YAML-GREEN-663`

## 2. 구조화 데이터 계산

`03-products.json`을 첨부하고 묻는다.

> 각 상품의 소계와 할인 적용 후 최종 금액을 계산하고 계산식을 보여줘.

예상 결과:

- 나루 라이트: 36,000원
- 나루 프로: 57,000원
- 나루 미니: 30,000원
- 할인 전: 123,000원
- 최종 금액: 120,000원

## 3. CSV 분석

`04-server-metrics.csv`를 첨부하고 묻는다.

> CPU 사용률이 가장 높은 시각과 수치를 찾고, warning 이후 정상 상태로 돌아온 시각을 알려줘.

예상 결과:

- CPU 최고점: 2026-07-23 09:20, 87%
- 정상 복귀: 2026-07-23 09:40

## 4. 여러 파일 동시 인식

`01-basic-korean.txt`, `02-meeting-notes.md`, `04-server-metrics.csv`를 동시에 첨부하고 묻는다.

> 첨부파일별 식별자를 모두 적어줘. 각 파일에서 날짜나 시각을 하나씩 찾아 파일명과 함께 표로 정리해줘.

세 식별자가 모두 답변에 있어야 한다:

- `MN-TXT-7429`
- `MN-MD-VIOLET-381`
- `MN-CSV-CYAN-904`

## 5. 후속 메시지 포함

`06-follow-up-memory.yaml`의 `후속 메시지에도 포함`을 선택하고 첫 메시지를 보낸다.

> 첨부파일을 읽었다면 문서 식별자만 답해줘.

응답이 끝난 뒤 파일을 다시 첨부하지 않고 묻는다.

> 앞서 첨부한 문서의 secret_phrase와 담당자 내선번호를 알려줘.

예상 결과:

- 비밀 문구: 보라색 등대는 새벽 세 시에 켜진다
- 내선번호: 7714

같은 시험을 새 대화에서 `후속 메시지에도 포함`을 끄고 수행하면 두 번째 질문에서 첨부 내용을 자동 참조하지 않아야 한다.

## 6. 첨부만 전송

`05-events.jsonl`만 선택하고 메시지 본문을 비운 채 전송한다. 전송 후 묻는다.

> 마지막 이벤트의 result와 reason을 알려줘.

예상 결과:

- result: denied
- reason: attachment is already linked
